# Logos de clientes

Sube aquí los logos de los clientes de Paisare.

## Formato requerido
- PNG con fondo transparente **o** SVG
- Orientación horizontal preferida
- Nombre de archivo: `nombre-cliente.png` (minúsculas, sin espacios, sin acentos)

## Ejemplos
- `oxxo.png`
- `desarrolladora-xyz.svg`
- `hotel-boutique.png`

Los logos se usarán en la sección `#clientes` de la landing page con filtro `grayscale` que se quita al hover.
