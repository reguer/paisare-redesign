# Paisare Redesign

Rediseño de plantilla para Paisare, usando como fuente de contenido la página actual de WordPress: https://www.paisare.com/

## Objetivo

Cambiar la presentación visual y la experiencia de la página sin perder los datos principales del sitio existente: servicios, métricas, portafolio, proceso de contratación, artículos, clientes y canales de contacto.

El primer entregable es un prototipo HTML autónomo:

- `Paisare Redesign.html`

## Enfoque

- Conservar el contenido comercial de la web actual.
- Hacer la página más rápida con una plantilla más ligera.
- Corregir o reemplazar funciones antiguas del tema WordPress cuando convenga.
- Separar decisiones, pendientes y alcance en archivos de documentación del proyecto.

## Documentación

- `CHANGELOG.md`: historial de cambios.
- `SCRATCHPAD.md`: notas vivas de trabajo.
- `EPICS_AND_STORIES.md`: épicas e historias de usuario.
- `ESTADO_ACTUAL.md`: estado actual del proyecto, riesgos y próximos pasos.
