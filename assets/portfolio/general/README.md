# Imágenes generales del portafolio

Esta carpeta contiene imágenes de apoyo que no pertenecen a un proyecto específico.
Úsala para: fondos, texturas, imágenes de proceso, fotos del equipo, material genérico.

## Convención de nombres
- `equipo-obra.jpg`
- `proceso-riego.jpg`
- `materiales-piedra.jpg`

Para imágenes de un proyecto concreto, crea una subcarpeta:
`assets/portfolio/[nombre-proyecto]/`
